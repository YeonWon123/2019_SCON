입력값으로 N이 주어지면, 1부터 N까지 다 해본다!
어떻게 하느냐?

ex) 198 => 198 + 1 + 9 + 8
0을 구하려면 : 198/1000000 = 0 = a
0을 구하려면 : 198/100000 = 0 = b
0을 구하려면 : 198/10000 = 0 = c
0을 구하려면 : 198/1000 = 0 = d
1을 구하려면 : 198/100 = a = e
9를 구하려면 : (198 - 100 * a) / 10 = f
8을 구하려면 : (198 - 100 * a - 10 * b) / 1 = g

for(int i = 1; i < N; i++) {
	if (N == i + a + b + c + d + e + f + g) {
		cout << i;
		return 0;
	}
}

cout << 0;
return 0;
